# Настройки Debian в WSL

После установки Debian в WSL (Windows Subsystem for Linux) рекомендуется выполнить базовые настройки для обновления системы и установки необходимых инструментов. Это обеспечит безопасность, стабильность и удобство работы. Все команды выполняются в терминале Debian (запустите `wsl -d Debian` или `debian` из Windows).

## 1. Обновление системы через apt
Apt — менеджер пакетов в Debian. Обновление системы включает синхронизацию списка пакетов и установку обновлений для безопасности и исправлений.

### Шаги:
1. Обновите список пакетов из репозиториев:
   ```
   sudo apt update
   ```

2. Обновите установленные пакеты до последних версий:
   ```
   sudo apt upgrade -y
   ```

3. (Опционально) Удалите ненужные пакеты и очистите кэш:
   ```
   sudo apt autoremove -y
   sudo apt autoclean
   ```

**Совет:** Запускайте `sudo apt update` и `sudo apt upgrade` регулярно. Если появится запрос на перезагрузку ядра, перезагрузите WSL командой `wsl --shutdown` из Windows.

## 2. Установка необходимых инструментов и настроек
Объединим процесс установки Git, Tree, 7-Zip и настройки автозавершения для удобства. Это включает обновление списка пакетов (если не делали ранее) и установку пакетов.

### Шаги:
1. Обновите список пакетов из репозиториев:
   ```
   sudo apt update
   ```

2. Установите Git (для контроля версий):
   ```
   sudo apt install git -y
   ```

3. Установите Tree (для отображения структуры каталогов в виде дерева):
   ```
   sudo apt install tree -y
   ```

4. Установите 7-Zip (для архивации и распаковки файлов):
   ```
   sudo apt install p7zip p7zip-full p7zip-rar -y
   ```

5. Установите bash-completion (для автозавершения параметров команд):
   ```
   sudo apt install bash-completion -y
   ```

6. Настройте автозавершение команд по истории (клавиши вверх/вниз):
   - Откройте файл на редактирование:
     ```
     sudo nano /etc/inputrc
     ```
   - Добавьте следующие строки в конец файла:
     ```
     set show-all-if-ambiguous On
     "\e[A": history-search-backward
     "\e[B": history-search-forward
     ```
   - Сохраните и выйдите из редактора (в nano: Ctrl+O, Enter, Ctrl+X).

7. Перезагрузите оболочку или выйдите и войдите снова в терминал, чтобы изменения вступили в силу.

**Совет:** Если возникнут проблемы, проверьте интернет-соединение или повторите `sudo apt update`. Для дополнительной информации используйте `man <команда>` (например, `man apt`).

Ниже расписаны по пунктам каждый устанавливаемый компонент с описанием, проверкой и примерами использования.

### Git
- **Описание:** Инструмент для контроля версий, позволяющий управлять изменениями в коде, работать с репозиториями (например, GitHub).
- **Проверка установки:** `git --version`
- **Примеры использования:**
  - Настройте имя и email: `git config --global user.name "Ваше Имя"` и `git config --global user.email "ваш.email@example.com"`
  - Клонируйте репозиторий: `git clone https://github.com/user/repo.git`

### Tree
- **Описание:** Утилита для отображения структуры каталогов в виде дерева, полезна для навигации по файлам.
- **Проверка установки:** `tree --version`
- **Примеры использования:**
  - Покажите структуру домашнего каталога: `tree /home`
  - Ограничьте глубину: `tree -L 2 /home`

### 7-Zip
- **Описание:** Мощный инструмент для архивации и распаковки файлов, поддерживающий форматы ZIP, RAR, 7Z и другие.
- **Проверка установки:** `7z --help`
- **Примеры использования:**
  - Распакуйте ZIP-архив: `7z x archive.zip`
  - Создайте 7Z-архив: `7z a archive.7z file.txt`
  - Распакуйте RAR: `7z x archive.rar` (требует `p7zip-rar`)

### Автозавершение команд и параметров
- **Описание:** Настройка для удобства в терминале: поиск команд по истории (клавиши вверх/вниз) и дополнение параметров с Tab.
- **Проверка:** Введите 2-3 символа команды и нажмите вверх — должна появиться предыдущая команда. Для Tab: введите `apt install ` и нажмите Tab — должны появиться варианты пакетов.
- **Примеры использования:**
  - Поиск по истории: введите `sudo` и нажмите вверх для повторного вызова предыдущих команд с `sudo`.
  - Дополнение параметров: `apt install <Tab>` покажет список пакетов. Если не работает, добавьте в `~/.bashrc`: `source /etc/bash_completion` и перезагрузите терминал.
