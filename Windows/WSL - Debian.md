# Настройки Debian в WSL

После установки Debian в WSL (Windows Subsystem for Linux) рекомендуется выполнить базовые настройки для обновления системы и установки необходимых инструментов. Это обеспечит безопасность, стабильность и удобство работы. Все команды выполняются в терминале Debian (запустите `wsl -d Debian` или `debian` из Windows).

## 1. Обновление системы через apt
Apt — менеджер пакетов в Debian. Обновление системы включает синхронизацию списка пакетов и установку обновлений для безопасности и исправлений.

### Шаги:
1. Обновите список пакетов из репозиториев:
   ```
   sudo apt update
   ```

2. Обновите установленные пакеты до последних версий:
   ```
   sudo apt upgrade -y
   ```

3. (Опционально) Удалите ненужные пакеты и очистите кэш:
   ```
   sudo apt autoremove -y
   sudo apt autoclean
   ```

**Совет:** Запускайте `sudo apt update` и `sudo apt upgrade` регулярно. Если появится запрос на перезагрузку ядра, перезагрузите WSL командой `wsl --shutdown` из Windows.

## 2. Установка: git, tree
Установите полезные инструменты для разработки и навигации по файлам.

### Шаги:
1. Установите Git (для контроля версий):
   ```
   sudo apt install git -y
   ```

2. Установите Tree (для отображения структуры каталогов в виде дерева):
   ```
   sudo apt install tree -y
   ```

3. Проверьте установку:
   ```
   git --version
   tree --version
   ```

**Совет:** После установки настройте Git (например, имя и email):  
```
git config --global user.name "Ваше Имя"
git config --global user.email "ваш.email@example.com"
```

Для Tree пример использования: `tree /home` — покажет структуру домашнего каталога.

## 3. Автозавершение команд и параметров
Настройте автозавершение для удобства работы в терминале: поиск команд по истории с клавишами вверх/вниз и дополнение параметров с помощью Tab.

### Шаги:

#### Автозавершение команд клавишами вверх/вниз
Суть настройки: вводите 2-3 первых символа команды и нажимайте клавишу вверх — система выводит полную команду, ранее выполненную и начинающуюся с этих символов.

1. Откройте файл на редактирование:
   ```
   sudo nano /etc/inputrc
   ```

2. Добавьте следующие строки в конец файла:
   ```
   set show-all-if-ambiguous On
   "\e[A": history-search-backward
   "\e[B": history-search-forward
   ```

3. Сохраните и выйдите из редактора (в nano: Ctrl+O, Enter, Ctrl+X).

4. Чтобы автозавершение начало работать, выйдите из консоли и подключитесь к ней снова.

**Совет:** Nano — простой редактор для начинающих; он показывает подсказки в нижней части экрана.

#### Параметры команд с помощью Tab
Изначально Tab дополняет только команду, но не параметры. Установите пакет для дополнения параметров.

1. Установите bash-completion:
   ```
   sudo apt install bash-completion -y
   ```

2. Перезагрузите оболочку или выйдите и войдите снова в терминал.

**Совет:** После установки Tab будет дополнять параметры команд (например, `apt install <Tab>` покажет доступные пакеты). Если не работает, добавьте в `~/.bashrc`: `source /etc/bash_completion` и перезагрузите терминал.

Если возникнут проблемы, проверьте интернет-соединение или повторите `sudo apt update`. Для дополнительной информации используйте `man <команда>` (например, `man apt`).
