# Включение гипервизора в виртуальной машине (VMware Workstation)

Позволяет запускать вложенную виртуализацию (nested virtualization), чтобы ВМ могла выступать в роли гипервизора и запускать свои собственные виртуальные машины.

Добавьте в файл .vmx:
```
hypervisor.cpuid.v0 = "FALSE"
mce.enable = "TRUE"
```

**Важно:**
- Требует поддержки VT-x/AMD-V на хосте и ВМ.
- Может снизить производительность; тестируйте на совместимость.
- Подходит для разработки/тестирования, но не для продакшена без проверки. Настройки ВМ перекрывают глобальные.
