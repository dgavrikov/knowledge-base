# Отключение файла памяти (vmem) в VMware Workstation

Файл подкачки (vmem) в папке виртуальной машины хранит часть ОЗУ. Его отключение:
- Снижает обращения к диску.
- Размещает всю ОЗУ ВМ в RAM хоста или его swap.

## В Windows

Добавьте в файл .vmx:
```
mainMem.useNamedFile = "FALSE"
```

**Важно:**
- Требует параметров: `prefvmx.minVmMemPct = "100"` и `MemTrimRate = "0"`.
- Без vmem невозможен suspend ВМ (нужен файл размером с настроенное ОЗУ).

## В Linux

Добавьте в файл .vmx: `mainMem.useNamedFile = "FALSE"`. Создаст временный swap-файл в temp-директории (удалится при выключении).

**Важно:** Настройки ВМ перекрывают глобальные (обычно).
