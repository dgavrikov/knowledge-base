# В VMware Workstation можно отключить использование файла памяти (vmem)

Файл подкачки, который находится в папке с виртуальной машиной и содержит часть оперативной памяти. Это позволяет: 
- уменьшить число обращений к диску;
- расположить всю оперативную память виртуальной машины в RAM хоста или его своп-пространстве.

## В Windows

В файле vmx в папке с виртуальной машиной нужно добавить параметр:
```
mainMem.useNamedFile = "FALSE".
``` 
Это отключит использование файла vmem. 

**Важно:**
- Параметр работает только если также установлены параметры: prefvmx.minVmMemPct = "100" и MemTrimRate = "0".
- Без файла vmem не будет возможен suspend виртуальной машины — для этого нужен файл с размером настроенной оперативной памяти.

## В Linux

В файле vmx в папке с виртуальной машиной нужно добавить параметр: mainMem.useNamedFile = "FALSE". **Однако для Linux** создастся временный файл подкачки во временной директории, который удалится при выключении виртуальной машины. 

**Важно:** настройки каждой из виртуальных машин перекрывают глобальные настройки, и в большинстве случаев настройки каждой из машин перекрывают настройки для всех.
