# Алгоритм проверки ИНН физического лица

ИНН — 12-значный номер налогоплательщика. Проверяет две последние контрольные цифры.

Шаги проверки (для цифр d1–d12):

1. Для 11-й цифры: вычислите сумму (d1×7 + d2×2 + d3×4 + d4×10 + d5×3 + d6×5 + d7×9 + d8×4 + d9×6 + d10×8 + d11×0) % 11. Если результат <10, 11-я цифра = результат; иначе =0.
2. Для 12-й цифры: сумма (d1×3 + d2×7 + d3×2 + d4×4 + d5×10 + d6×3 + d7×5 + d8×9 + d9×4 + d10×6 + d11×8 + d12×0) % 11. Если результат <10, 12-я цифра = результат; иначе =0.

**Важно:** Алгоритм для физических лиц (12 цифр). Для юридических лиц (10 цифр) — аналогично, но одна контрольная цифра с коэффициентами 2,4,10,3,5,9,4,6,8,0.
