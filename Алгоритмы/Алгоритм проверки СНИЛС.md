# Алгоритм проверки СНИЛС

СНИЛС проверяется аналогично ИНН. Формат: NNN-NNN-NNN ZZ (N — номер, ZZ — контрольное число).

Шаги проверки:
1. Умножьте каждую цифру номера на позицию справа (первая цифра ×9, вторая ×8 и т.д.).
2. Суммируйте произведения.
3. Проверьте сумму:
   - Если <100, она должна равняться ZZ.
   - Если =100 или 101, ZZ должно быть 00.
   - Если >101, разделите сумму на 101; остаток проверьте по пп. 1–2.

**Важно:** Алгоритм работает только для номеров >001-001-998.
