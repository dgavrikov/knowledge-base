# Инструкция по установке плагинов в Sublime Text 4

Для удобного управления плагинами в Sublime Text 4 рекомендуется использовать **Package Control** — официальный менеджер пакетов.

## Шаг 1. Установка Package Control

1. Откройте Sublime Text 4.
2. Нажмите `Ctrl + `` (тильда)`, чтобы открыть консоль.
3. Перейдите на сайт https://packagecontrol.io/installation и скопируйте код для Sublime Text 4.
4. Вставьте код в консоль Sublime Text и нажмите Enter.
5. Перезапустите Sublime Text.

## Шаг 2. Установка плагинов через Package Control

1. Нажмите `Ctrl + Shift + P`, чтобы открыть палитру команд.
2. Введите `Package Control: Install Package` и выберите эту команду.
3. В появившемся списке начните вводить название плагина и выберите нужный из списка.
4. Повторите для каждого плагина из списка ниже:

- **CoolBase64** — удобное кодирование и декодирование Base64.
- **Markdown Extended** — расширенные возможности работы с Markdown.
- **Pretty JSON** — форматирование и подсветка JSON.

## Готово!

Теперь плагины установлены и готовы к использованию.
