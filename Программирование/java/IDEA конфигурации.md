## Ограничение проекта по памяти

Для запуска Spring Boot проекта в IntelliJ IDEA с ограничением использования оперативной памяти до 512 МБ, необходимо настроить параметры JVM.

Шаги:

1. Откройте настройки запуска проекта.
2. Перейдите в раздел Configuration.
3. В поле VM options добавьте параметры -Xms512m -Xmx512m.

Пример настроек:

```xml
-Xms512m -Xmx512m
```

Эти параметры устанавливают начальный размер кучи (`-Xms`) и максимальный размер кучи (`-Xmx`) равными 512 МБ.
